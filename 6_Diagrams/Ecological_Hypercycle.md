# Ecological Hypercycle Diagram — Specification

## 1. Components
- Microbial Biomass (N)
- Enzyme Cascades (v1, v2, v3)
- Substrate Pools (S1, S2, S3)
- Carbon Flux (C)
- Nitrogen Flux (N2)
- Soil Moisture (W)
- Feedback Arrows

## 2. Relationships
- N → Enzyme Cascades
- Enzyme Cascades → Substrate Turnover
- Substrate Turnover → Carbon Flux
- Carbon Flux → Microbial Biomass
- Nitrogen Flux ↔ Microbial Biomass
- Moisture → Microbial Growth
- Feedback loops connecting all components

## 3. Layout
Circular hypercycle with arrows connecting each component.

## 4. Color Coding
- Microbes: green
- Enzymes: yellow
- Substrates: blue
- Carbon flux: brown
- Nitrogen flux: red
