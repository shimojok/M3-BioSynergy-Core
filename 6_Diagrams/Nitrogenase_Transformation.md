# Nitrogenase Transformation Diagram — Specification

## 1. Components
- N2 (atmospheric nitrogen)
- Nitrogenase Complex
- Electrons (e-)
- ATP
- NH3 (ammonia)
- H2 (byproduct)
- Soil Nitrogen Pool

## 2. Relationships
- N2 + e- + ATP → Nitrogenase → NH3 + H2
- NH3 → Soil Nitrogen Pool

## 3. Layout
Left: Inputs  
Center: Nitrogenase  
Right: Outputs

## 4. Color Coding
- N2: blue
- Nitrogenase: red
- Electrons: yellow
- ATP: purple
- NH3: green

## 5. Style Notes
- Enzyme complex drawn as a catalytic “core”
